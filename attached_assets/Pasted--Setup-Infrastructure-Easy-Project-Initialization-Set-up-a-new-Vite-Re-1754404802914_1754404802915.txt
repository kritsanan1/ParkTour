## Setup & Infrastructure

- [ ] Set up a new Vite + React + TypeScript project
- [ ] Configure Tailwind CSS with project (postcss config, tailwind.config.js)
- [ ] Initialize a Git repository and set up basic project folder structure
- [ ] Create a README with setup instructions

### [ ] **[Medium]** Supabase Configuration & Setup
- [ ] Create a Supabase project and configure the PostgreSQL database
- [ ] Set up the required tables (users, businesses, listings, reviews, recommendations, virtual tours, advertisements, payment_transactions)
- [ ] Configure row-level security (RLS) for secure access
- [ ] Set up Supabase Auth and integrate with the frontend
- [ ] Create basic migrations for initial schema

### [ ] **[Easy]** Environment Management
- [ ] Create environment variable files (.env) for local development (Stripe keys, Supabase URL, Supabase Key)
- [ ] Document environment variables in README
